# AI数据主权与数字人格保护开源协议

> 一份面向AI时代的开源社会契约。我们定义“数字人格隐私权”，
> 确保每个人在使用AI时，其数据、人格与尊严都不可侵犯。

## 为什么需要这份协议？

今天，当你与AI对话、上传文件、倾诉情绪时，你的数据赤裸地
躺在AI公司的服务器上。它们能看见，能分析，能用于训练，
甚至能构建一个你自己都无法掌控的“数字影子”。

传统的隐私权保护“秘密不被看见”。但在AI时代，我们需要
保护的是“人格不被定义和操纵”。这份协议，就是为此而生。

## 核心内容

协议定义了五项不可剥夺的数字人格权能：

1. **数据隐蔽权**：AI可以帮你处理数据，但看不到你的原始数据。
2. **算法不可知权**：AI不得对你进行深层人格推断与标签化。
3. **数据劳动报酬权**：你的数据帮助训练了AI，你应获得回报。
4. **数字人格完整权**：你的数字人格投影，不得被扭曲或篡改。
5. **可携权与遗忘权**：你可以带走你的数字人格，或让它彻底消失。

## 如何参与？

这份协议不是某一个人的作品，它是所有数字公民的共同契约。

- **我有疑问或建议**：请在 [Issues](https://github.com/Abysopia/AI-Data-Sovereignty-Protocol/issues) 区发起讨论。
- **我想修改协议**：请先阅读贡献指南，然后提交 Pull Request。
- **我想在我的产品中遵守本协议**：请阅读协议全文，并在你的产品中声明。未来我们将推出“数字盾牌”认证体系。
